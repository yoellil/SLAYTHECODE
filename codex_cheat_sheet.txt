# CODEX Language Cheat Sheet

## Language Name
- **CODEX**

## Data Types
- `bolt` (or `digit`): Numeric type (integer). Used for damage, defense, healing, strength, and any arithmetic.
- `scroll`: String type. Used for text.
- `char`: Character type. A single character.
- `flag`: Boolean type (`true` / `false`). Used for logical conditions.
- `dialogue`: A special string type used for the two-turn `speak` mechanic.

## Variables & Assignment
- **Assignment Operator:** `~`
- **Syntax:** `datatype identifier ~ expression.`
- Every statement must end with a period (`.`).

### Battle-Relevant Variable Names
The compiler only recognizes these specific variable names to affect combat:
- `dmg` or `damage`: How much damage your snippet deals.
- `def`: How much block (defense) you gain.
- `heal`: How much HP you restore.
- `pow`: How much strength you gain.

## Operators
- **Arithmetic:** `+` (add), `-` (subtract), `*` (multiply), `/` (divide)
- **Comparison:** `==` (equal), `!=` (not equal), `<` (less than), `>` (greater than), `<=` (less than or equal), `>=` (greater than or equal)

## Control Flow
### Conditional Statements (`TEST`)
- **Syntax:** `TEST condition FAIL statement PASS statement CLOSE.`
- Executes the `PASS` statement if the `condition` is true, otherwise executes the `FAIL` statement.
- **Example:**
  `TEST 1 < 2 FAIL speak "Logic is broken.". PASS speak "Logic is sound.". CLOSE.`

### Loop Statements (`REPEAT`)
- **Syntax:** `REPEAT condition UNTIL statement BREAK.`
- Executes the `statement` as long as the `condition` is true.
- **Example:**
  `REPEAT true UNTIL digit dmg ~ 4. BREAK.`

## Actions
### `speak`
- Sends a message and attempts to persuade the enemy.
- **One-Turn Persuasion:**
  - `speak "We can walk away from this.".`
- **Two-Turn Dialogue:**
  1. **Turn 1 (Prepare):** Declare a `dialogue` variable. This prepares the dialogue.
     - `dialogue myWords ~ "This fight is meaningless.".`
  2. **Turn 2 (Execute):** A special "Speak" card will be added to your hand. Use it to deliver the dialogue. The code on that card will be:
     - `speak myWords.`

## Common Snippet Patterns
- **Basic Attack:**
  `bolt dmg ~ 6.`

- **Stronger Attack with Arithmetic:**
  `bolt dmg ~ 10 + 5.`

- **Basic Block:**
  `bolt def ~ 8.`

- **Healing:**
  `bolt heal ~ 10.`

- **Strength Buff:**
  `bolt pow ~ 3.`
